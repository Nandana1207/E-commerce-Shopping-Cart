# 🛍️ E-commerce Shopping Cart

This is a simple web application built using Flask.

## 🔥 Features
- View products
- Add to cart
- Calculate total price

## 🛠️ Tech Used
- Python
- Flask
- HTML, CSS

## ▶️ How to Run
1. Install Flask: pip install flask
2. Run: python app.py
3. Open browser: http://127.0.0.1:5000/
